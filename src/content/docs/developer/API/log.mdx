title: Log

**Docs:** https://docs.api.kunkun.sh/interfaces/index.ILogger

This log API not only logs to browser console, but also logs to a log file stored on disk.
Could be useful for debugging.
If there is any serious error, use this log API and display a toast/notification message to the user.

```ts
log.debug("debug message")
log.error("error message")
log.info("info message")
log.trace("trace message")
log.warn("warn message")
```
