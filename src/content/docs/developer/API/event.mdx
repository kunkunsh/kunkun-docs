Event API contains commands to listen to some window/app events.

For more details, refer to [**Docs**](https://docs.api.kunkun.sh/interfaces/index.IEvent).

## Sample Code

import ImportCodeBlock from '@/components/api/import-code-block.astro'

```ts
event.onDragDrop((payload) => {
	const filePaths = payload.paths
	const { x, y } = payload.position
})

event.onWindowFocus(() => {
	document.getElementById("search-bar")?.focus()
})

event.onWindowBlur(() => {
  // e.g. disconnect from server
})
```
